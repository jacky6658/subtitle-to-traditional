# AIJobvideo — 字幕轉繁體工具

> 自動辨識影片簡體字幕，一鍵轉換為繁體中文並嵌回影片。  
> 所有運算在本機執行，影片不會上傳至任何伺服器。

---

## 功能

- 自動語音辨識（Whisper）並生成字幕
- 簡體 → 繁體中文轉換
- 自動偵測原字幕位置並覆蓋
- 轉換完成可預覽並下載 MP4
- 同時輸出 `.srt` 字幕檔
- 一鍵自動更新

---

## 系統需求

| 項目 | 需求 |
|------|------|
| 作業系統 | Windows 10/11、macOS 12+ |
| Python | 3.10 – 3.12（Windows 版會自動安裝） |
| 磁碟空間 | 約 3 GB（含 Whisper 模型） |
| 影片長度 | 建議不超過 1 分鐘（測試用） |

---

## 安裝與啟動

### Windows

1. 下載本 repo（右上角 **Code → Download ZIP**），解壓到任意資料夾
2. 雙擊 `start.bat`
3. 首次執行會自動安裝 Python 套件（約 5–10 分鐘）
4. 安裝完成後瀏覽器自動開啟，桌面自動建立 **AIJobvideo** 捷徑
5. 之後每次直接點桌面捷徑即可

### macOS

1. 下載並解壓，或直接使用 `.app` 應用程式
2. 雙擊 `啟動.command`（若提示無法開啟，右鍵 → 打開）
3. 瀏覽器自動開啟工具

---

## 使用方式

1. 開啟工具後點選「**字幕轉繁體工具**」
2. 拖曳影片（MP4 / MOV / MKV / AVI）或點擊上傳
3. 點擊「**開始轉換**」
4. 等待處理完成（首次需下載 Whisper 模型）
5. 預覽無誤後點擊「**下載**」

---

## 更新

主頁面點擊「**檢查更新**」→「**立即更新**」，工具會自動下載最新版本並重啟，無需手動操作。
